# Contributor native languages — a COMMITTED signal of which native languages
# this repository's contributors use. The SessionStart hook
# (hooks/session-start.sh) reads it to write operator-facing Clairvoyance
# handoffs in the native language of whoever is driving the current session.
#
# This file is committed on purpose: it is the only per-contributor source that
# survives a volatile/ephemeral checkout (Claude web, CI), and it records the
# repository's contributor-language signal rather than fixing one owner's
# language on everyone.
#
# Format: one `identity = language` line per contributor, keyed by git email or
# git name. Keys match case-insensitively; `#` comments and blank lines are
# ignored. A contributor can also override per session with the
# CLAIRVOYANCE_OPERATOR_LANGUAGE environment variable instead of adding a line.
#
# Privacy: this file is committed, so do NOT put a personal email here — it would
# be a public scraping target. Use a non-harvestable identity instead: a GitHub
# `users.noreply.github.com` address, or just your git name.
#
# Example:
#   Alice = English
#   31282861+tvna@users.noreply.github.com = Japanese
